# 🎵 Music Player

A simple and interactive web-based music player built using HTML, CSS, and JavaScript. This project demonstrates core concepts of audio handling, DOM manipulation, and user interaction in a clean and minimal interface.

---

## 🚀 Features

* ▶️ Play / Pause songs
* ⏭ Next / Previous track navigation
* 🎚 Audio playback control
* 📱 Responsive UI (basic)
* 🎧 Smooth user interaction

---

## 🛠 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript (Vanilla)** – Functionality & logic

---

## 📂 Project Structure

```
Music-Player/
│── index.html
│── style.css
│── script.js
│── /songs
│── /assets
```

---

## ⚙️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/mohan2-hash/Music-Player.git
   ```

2. Navigate to the project folder:

   ```
   cd Music-Player
   ```

3. Open `index.html` in your browser.
